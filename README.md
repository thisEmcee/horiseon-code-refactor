# Horiseon Code Refactor

## Description

This project is a code refactore for the Horiseon homepage. The purpose of this refactor is to make the code more effecient, comprehensible, and in turn more accessible.

This changes provided does not change the displayed styling, only the underlying code.

### Tasks Complete

* Redudant classes consolidated
* HTML elements replace with semeantic HTML
* Alt text added to images
* Comments added to provide context
* CSS elements consolidated and structured

### Links

[Link to the deployed website](https://thisemcee.github.io/horiseon-code-refactor/) \

## Installation

Upload index.html, liscense.txt, and assets folder to desired webserver. The assests folder contains the styling (css files) and images.

## Usage

Once uploaded and deployed the application can be viewed in the desktop browser. 
Not optimized for mWeb or native.
